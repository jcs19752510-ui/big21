### 회원 관리 프로그램 (Member Management System)

파이썬(Python)과 pickle 모듈을 활용하여 구현한 콘솔 기반 회원 관리 프로그램입니다.

회원 정보는 로컬 파일(members.dat)에 안전하게 저장되며,

동명이인 처리, 입력값 유효성 검증, 데이터 영속성 기능을 포함하고 있습니다.

### 📌 프로젝트 개요

* ✔ 콘솔 환경에서 동작하는 직관적인 회원 관리 프로그램

* ✔ pickle을 이용한 로컬 데이터 저장 및 불러오기 지원

* ✔ 이름, 전화번호, 주소, 회원 구분 정보를 체계적으로 관리

* ✔ 동명이인 선택 처리 및 입력값 검증 기능 포함

### ✨ 주요 기능

### 1. 회원 등록 (Create)

아래 정보를 입력받아 회원을 등록합니다.

* 이름: 1~5자 이내

* 전화번호: 010으로 시작하는 11자리 숫자

* 주소: 자유 형식 입력 가능

* 구분(div): 가족 / 친구 / 기타 중 선택

### 2. 회원 조회 (Read)

현재 저장된 전체 회원 목록과 총 회원 수를 한눈에 확인할 수 있습니다.

### 3. 회원 수정 (Update)

회원 이름으로 검색 후 주소, 전화번호 등의 정보를 수정할 수 있습니다.

* 동명이인이 존재할 경우 수정할 회원을 선택할 수 있습니다.

### 4. 회원 삭제 (Delete)

등록된 회원 정보를 시스템에서 삭제합니다.

### 5. 데이터 영속성

pickle 모듈을 사용하여 프로그램 종료 후에도 회원 데이터가 members.dat 파일에 유지됩니다.

### 🛠 요구 사항

* Python 3.x 이상 (권장: Python 3.12)

### 🚀 실행 방법

터미널(또는 VS Code PowerShell)을 열고 아래 순서대로 실행합니다.

1. VS Code 실행

2. 터미널(PowerShell) 열기

3. 프로젝트 디렉토리로 이동

4. 프로그램 실행

### 📂 프로젝트 구조 예시

### 📑 회원 데이터 스키마

회원 정보는 Dictionary 형태로 관리됩니다.

### ⚙ 기술 스택 및 데이터 구조

| 항목              | 내용                     |
| --------------- | ---------------------- |
| Language        | Python 3.12            |
| Storage (Local) | pickle (바이너리 파일 저장 방식) |
| Data File       | members.dat            |

### ✅ 유효성 검증(Validation) 규칙

프로그램의 안정성을 위해 입력 단계에서 다음 규칙을 검증합니다.

### 1. 이름(name)

* 공백을 제외하고 1자 이상 5자 이내여야 합니다.

### 2. 전화번호(phone)

* 반드시 010으로 시작해야 합니다.

* 총 11자리 숫자여야 합니다.

* 정규표현식(re)을 이용해 검증합니다.

### 3. 회원 구분(div)

* 가족 / 친구 / 기타 중 하나만 입력 가능합니다.

* 허용되지 않은 값이 입력되면 재입력을 요청합니다.

### 💾 데이터 저장 방식

회원 데이터는 pickle 모듈을 통해 바이너리 형태로 저장됩니다.

* 프로그램 실행 시 members.dat 파일에서 데이터를 불러옵니다.

* 회원 추가, 수정, 삭제 시 즉시 파일에 저장됩니다.

* 프로그램 종료 후에도 데이터가 유지됩니다.
